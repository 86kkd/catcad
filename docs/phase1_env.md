# 第一阶段：蛇形布线环境

目标：在网格上进行单网络、单层布线，具有近似设计规则检查(DRC)。

观察空间 (高×宽×通道):
- 障碍物
- 目标场（归一化曼哈顿距离）
- 层级独热编码（单层时恒为1）
- 占用情况（已布线路径）
- DRC裕度（1表示安全，0表示禁止）

 动作空间:
 - 上/下/左/右
 - 左上/左下/右上/右下（禁止穿角，需两相邻正交单元可行）
 - 过孔（占位符，单层时禁用）
 - 停止

状态转移:
- 每步移动一个单元格；记录占用
- 验证边界/禁区/自交叉
- 转角时施加惩罚
- 基于到目标距离的奖励整形

 奖励设置（默认值）:
- 成功 +10
- 每步 −0.01×步长（斜向≈√2）
- 转角 −0.2×转角归一化（45° 为 0.5，90° 为 1）
- 违规 −1.0
- 接近度整形 −0.01×曼哈顿距离

终止条件:
- 到达目标
- 达到最大步数
- 手动停止
